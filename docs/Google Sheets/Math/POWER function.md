# POWER function

Returns a number raised to a power.

## Sample Usage

`POWER(4,0.5)`

`POWER(A2,B2)`

`POWER(2,5)`

## Syntax

`POWER(base, exponent)`

* `base` - The number to raise to the `exponent` power.
* `exponent` - The exponent to raise `base` to.
