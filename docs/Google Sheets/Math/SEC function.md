# SEC function

The SEC function returns the secant of an angle measured in radians.

## Parts of a SEC formula

`SEC(angle)`


| **Part** | **Description**                                       |
| -------- | ----------------------------------------------------- |
| `angle`  | The angle to find the secant of, measured in radians. |

## Sample formulas

`SEC(3.14)`

`SEC(A1)`

`=SEC(1)`

## Examples


|       | **A**       | **B**        |
| ----- | ----------- | ------------ |
| **1** | **Formula** | **Result**   |
| **2** | =SEC(1)     | 1.850815718  |
| **3** | =SEC(-1)    | 1.850815718  |
| **4** | =SEC(4)     | -1.529885656 |
| **5** | =SEC(0)     | 1            |
