# RAND

Returns a random number between 0 inclusive and 1 exclusive.

### Sample Usage

`RAND()`

### Syntax

`RAND()`

### See Also

[`RANDBETWEEN`](https://support.google.com/docs/answer/3093507): Returns a uniformly random integer between two values, inclusive.
