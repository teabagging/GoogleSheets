# DEGREES

The DEGREES function converts an angle value in radians to degrees.

### Sample Usage

`DEGREES(PI())`

`DEGREES(6)`

`DEGREES(A2)`

### Syntax

`DEGREES(angle)`

* `angle` - The angle to convert from radians to degrees.
